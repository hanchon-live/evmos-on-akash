# Evmos on AKT

## Build

```sh
docker build . -t elhachon/evmos-on-akash:v6.0.1-r2
```

## Push

```sh
docker push elhachon/evmos-on-akash:v6.0.1-r2
```

## Usage

```sh
docker pull elhachon/evmos-on-akash
# or
git clone https://github.com/hanchon-live/evmos-on-akash.git
cd evmos-on-akash
docker-compose up
```
